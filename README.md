# PDF Tools

**25 PDF tools in one HTML file — no uploads, no server, no account.**

Everything runs 100% in your browser. Your files never leave your device.

🔗 **[https://abdullahmuhy-pixel.github.io/Pdf-tools)**

---

## Tools

### Create & Convert
| Tool | Description |
|------|-------------|
| 🖼️ Image → PDF | Convert JPG, PNG, WEBP images to PDF |
| 📷 Scan to PDF | Use your camera to scan documents |
| 📝 Text to PDF | Convert plain text to a formatted PDF |
| 🌐 HTML to PDF | Convert HTML files or pasted HTML to PDF |
| 📄 Word to PDF | Convert .docx files (headings, lists preserved) |
| 🖼️ PDF to JPG | Export each PDF page as a JPG image |
| 📋 PDF to Text | Extract all readable text from a PDF |
| 🔍 OCR | Extract text from scanned PDFs or images (7 languages) |

### Organise PDF
| Tool | Description |
|------|-------------|
| 🔗 Merge PDF | Combine multiple PDFs into one |
| ✂️ Split PDF | Extract page ranges or split into individual pages |
| 📑 Organize PDF | Drag-and-drop page reordering |
| 🔄 Rotate PDF | Rotate all or specific pages |
| 🗑️ Delete Pages | Select and remove pages |
| 📄 Insert Page | Insert blank pages at any position |
| ✂️ Crop PDF | Trim page margins |

### Optimize & Content
| Tool | Description |
|------|-------------|
| 📦 Compress PDF | Reduce file size |
| 💧 Watermark | Text or image watermark on every page |
| 🔢 Page Numbers | Add page numbers (6 positions, 4 formats) |
| ✏️ Sign PDF | Click-to-place text + draw/queue signatures |
| ⬛ Redact PDF | Permanently black out sensitive content |
| ℹ️ Edit Metadata | Edit title, author, subject, keywords |
| 📝 Fill Forms | Fill and flatten PDF form fields |

### Security
| Tool | Description |
|------|-------------|
| 🔐 Protect PDF | Remove password from encrypted PDFs |
| ✍️ Signature | Draw, type, or upload a signature (PNG/PDF) |

## Features

- ✅ **100% offline** — works without internet after first load
- ✅ **Installable PWA** — add to home screen on Android/iOS/desktop
- ✅ **No file size limits** — process large files locally
- ✅ **Touch support** — drag-and-drop works on mobile
- ✅ **No tracking** — no analytics, no cookies, no ads
- ✅ **Open source** — MIT licence

## Libraries used

| Library | Purpose |
|---------|---------|
| [pdf-lib](https://pdf-lib.js.org/) | PDF creation and manipulation |
| [PDF.js](https://mozilla.github.io/pdf.js/) | PDF rendering and text extraction |
| [Mammoth.js](https://github.com/mwilliamson/mammoth.js) | DOCX to HTML conversion |
| [Tesseract.js](https://tesseract.projectnaptha.com/) | OCR text recognition |

## Deploy your own

### GitHub Pages (recommended)
1. Fork this repo
2. Go to **Settings → Pages**
3. Source: **Deploy from branch → main → / (root)**
4. Your site will be live at `https://yourusername.github.io/pdf-tools`

### Local use
Just open `index.html` in any modern browser. No server needed.
